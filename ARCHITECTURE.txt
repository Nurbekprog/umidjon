2. CURSOR TIZIMI — macOS STIL (ALOHIDA MUHIM)
═══════════════════════════════════════════════════════════════

ARXITEKTURA:
  cursor-provider.tsx  ->  Global Context (variant state, setCursor fn)
  mac-cursor.tsx       ->  SVG render + GSAP quickTo harakat
  cursor-types.ts      ->  CursorVariant union type

VARIANT -> SVG XARITASI:
  "arrow"    -> /Cursors/Cursor/Arrow.svg      (default holatda)
  "pointer"  -> /Cursors/Cursor/Pointer.svg    (a, button ustida)
  "grab"     -> /Cursors/Cursor/Grab.svg       (card, slider ustida)
  "grabbed"  -> /Cursors/Cursor/Grabbed.svg    (drag jarayonida)
  "text"     -> /Cursors/Cursor/Text.svg       (p, h1, input ustida)
  "zoom-in"  -> /Cursors/Cursor/ZoomIn.svg     (img ustida)
  "move"     -> /Cursors/Cursor/Move.svg       (draggable element)

ISHLASH TARTIBI:
  1. CursorProvider app/layout.tsx da wrap qiladi
  2. mac-cursor.tsx — GSAP quickTo bilan sichqonchaga ergashadi
  3. Har qanday komponentda: const { setCursor } = useCursor()
  4. Ishlatish:
       onMouseEnter={() => setCursor("pointer")}
       onMouseLeave={() => setCursor("arrow")}
  5. Mobilda (useMediaQuery max-width:768px) — cursor render bo'lmaydi

MAC-CURSOR.TSX TEXNIK TALABLAR:
  - position: fixed, top:0, left:0, pointer-events:none, z-index:9999
  - GSAP quickTo: x va y alohida (xQuick, yQuick — performance)
  - SVG o'lchami: 32x32px default, transform-origin: top-left
  - Silliq harakat: duration:0.15 (arrow), 0.1 (pointer), 0.2 (grab)
  - globals.css da: * { cursor: none !important }
  - dynamic(()=>import(), {ssr:false}) bilan yuklanadi

cursor-types.ts:
  export type CursorVariant =
    | "arrow" | "pointer" | "grab" | "grabbed"
    | "text" | "zoom-in" | "zoom-out" | "move" | "help"

  - DOM faqat useRef orqali (document.querySelector MUMKIN EMAS)
  - gsap.context() bilan cleanup — har doim

[CURSOR]
  - globals.css: * { cursor: none !important }
  - mac-cursor.tsx dynamic(ssr:false) bilan
  - Mobilda cursor-provider null qaytaradi

[THREE.JS]
  - HeroCanvas faqat dynamic(ssr:false)
  - Suspense fallback: animatsiyali div
  - useFrame ichida yangi obyekt yaratilmaydi

[KOMPONENTLAR]
  - Har fayl maksimum 150 qator
  - "use client" faqat zarur joylarda
  - Barcha import @/ alias bilan
  - components/ root darajasida hech qanday .tsx yo'q

[STYLING]
  - Spacing faqat Tailwind rem
